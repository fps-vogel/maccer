māccer

A tool for macronizing Latin text.

Download this project as a .zip file Download this project as a tar.gz file

N.B. This tool is still in development. The word key is under revision, and the full offline version of the program, which can handle longer texts and includes all data and source code, is not available yet for download since I am still writing documentation for it.

Instructions

  1. Paste or write plain Latin text into the box above.
  2. Click māc!.
  3. Insert macrons where missing (✖) or uncertain (✒, ✪, ❡) by placing a hyphen (-) after each long vowel, then click a- › ā.
  4. Remove flags by clicking flags.

To contact me: Click ✉. If you have Latin text with macrons and would like to make this tool smarter, please let me know!

To report a problem: Select your text, click ✉, send, and I will look into it. If your problem is with anything other than macronize, please make a note of it in the email.
FLAGS
✖ unknown word.
✒ ambiguous: uncertain vowels marked with a tilde (~).
✪ guessed based on frequency.
❡ prefix or enclitic detected attached to a known word.
☛ invalid characters detected.

Tips

  • Text can be de-macronized by clicking ō › o.
  • If any text is selected, commands will be applied only to the selection.
  • Click ↺ to undo the most recent command, but this will also undo all changes since then. Click it again to redo the command and all changes after it.
  • Use comments (C-style // or /* */) to exclude any words that should not be macronized, e.g. a passage in another language.
  • Markdown formatting is supported. Click the preview icon in the text area to render.

Q & A

I can’t copy text from Word without losing all the formatting!

Web applications have trouble reading Word formatting, so here it is not supported. But most formatting can be preserved by following these steps:

  1. Plug your Word file into the Word to Markdown Converter.
  2. Copy the resulting Markdown-formatted text, paste it above, and macronize it.
  3. Generate formatting via the preview button, or for more customizability, via Pandoc.

This program is really slow!

I’m not a programmer, least of all a web programmer: I did not imagine Maccer running on a webpage until late in its development, when I found out Lua scripts could be run on the web via the Lua VM. I thought it would be worth a try, and here is the result. It would be faster if it were ported to pure JavaScript, but it is what it is. For best performance, download the full desktop version via one of the download links above—the .zip version if you’re not sure.

This program doesn't work on [insert book title here]!

The web version simply cannot macronize texts of over 2000 words. The offline version has no such limitation.

How does it work?

Maccer operates based on a list of Latin word forms generated from digital texts already marked with macrons. Currently it does not generate inflected forms based on principal parts. This has the advantage of not inventing forms that were not actually used, but also the disadvantage of knowing only as many forms as occur in the source texts.

Besides finding definite matches between plain- and macron-forms, Maccer can handle a few special cases: in addition to the ones denoted by flags, abbreviations are detected, e.g. “a.” in “a. Kal.” remains short but “a” not preceding a period becomes long, as in “ā Caesare”.

But these online texts with macrons are full of errors!

Yes, the texts I have found that have vowel quantity marked range from nearly consistent to worse than nothing, and it would be impossible to correct every single mistake in them. However, with the help of a script that detects possible errors in the word key, and some human intervention, this automated process becomes much less haphazard. See the documentation in the offline version.

That being said, this tool is not and does not pretend to be a substitute for the human editor, only to make this particular task less time-consuming and, I hope, less tedious.

My hope is that with the help of this tool, we can more easily mark texts with macrons and become even better readers and speakers of Latin.
